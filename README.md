# CloverPit Archipelago

A **multiworld randomizer mod** designed to work with the **Archipelago** randomizer system.

This mod allows players to randomly distribute items and objectives across multiple players in a cooperative randomizer experience.

---

## Installation

To get started, follow these steps:

1. **Install BepInEx** (if not already installed).
    - You can download it from the [BepInEx GitHub](https://github.com/BepInEx/BepInEx).

2. **Download and Install the Mod**:
    - **Thunderstore**: Download the mod from [Thunderstore](https://thunderstore.io/c/clover-pit/p/Syntheal/CloverPit_Archipelago/).
    - **Using a Mod Manager**: We recommend using [r2modman](https://github.com/ebkr/r2modmanPlus) to install and manage mods easily.
    - **Manual Installation**:
        1. Extract the mod files into the `BepInEx/plugins/` folder.

3. **Launch the Game**:
    - Once everything is set up, launch the game and the mod will be active.

---

## Configuration

This mod is designed to work with the **Archipelago multiworld randomizer**. You will need to set up an Archipelago server and configure it to use this mod. For detailed setup and instructions, refer to the [Archipelago website](https://archipelago.gg).

### Key Configuration Points:
- **Archipelago Server**: Follow the server setup instructions on the Archipelago website.

---

## Dependencies

To run this mod, you'll need:

- **BepInEx**: Ensure you have the correct version of BepInEx installed for your game.
- **Archipelago**: You must be connected to an Archipelago multiworld server. For details on how to set this up, visit [Archipelago.gg](https://archipelago.gg).

## Troubleshooting

If you're experiencing issues, here are some common steps to troubleshoot:

1. **Ensure BepInEx is installed correctly**: Sometimes the mod may not work if BepInEx isn't installed or configured correctly.
2. **Check your Archipelago configuration**: Make sure that the multiworld server is set up properly and that the mod is configured to work with it.
3. **Mod conflicts**: If you're using other mods, try disabling them to see if they conflict with CloverPit Archipelago.
